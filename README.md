# Disaster Aid

## Tech Stack

1. MongoDB
2. Node.js
3. Express
4. EJS

## How to run

1. Clone the repo

```sh
git clone https://github.com/Rinso123/Disaster-Aid.git
```

2. Install dependencies

```sh
npm install
```

3. Update the `.env` file

   1. Copy the .env.example file to .env
   1. Update the values in the .env file

4. Start the server

```sh
npm run dev
```
